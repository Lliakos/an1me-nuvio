## 🛠️ Installation (Plugin / Add-on)

To use this provider, you do not need to download or compile any code. You simply need to add the raw manifest link directly into your media application as a plugin.

1. **Copy** the following raw manifest URL:
   https://raw.githubusercontent.com/Lliakos/an1me-nuvio/refs/heads/main/manifest.json

2. **Open** your preferred media aggregation app (e.g., Nuvio, Stremio).
3. Navigate to the **Add-ons**, **Plugins**, or **Extensions** menu.
4. **Paste** the copied URL into the search bar or the designated "Add URL" field.
5. Click **Install** and accept any prompts to add the provider to your setup.
